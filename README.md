# Czyszczenie-i-analiza-danych-sprzedazowych
Projekt polega na czyszczeniu i analizie danych sprzedazowych w języku Python.
Główne funkcjonalności:
*Automatyczne wyłonienie top 5 najlepiej sprzedających się produktów.
*Obliczenie całkowitego przychodu oraz średniej wartości sprzedaży każdego produktu.
*Zestawienie popularności transakcji gotówkowych oraz bezgotówkowych.
*Ranking miesięcy ze względu na wygenerowany przychód.

Wykorzystane technologie:
Python (Pandas, Openpyxl)
